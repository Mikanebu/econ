About

The Open Economics project provides open content, data and code related to Economics. This site itself provides interfaces to some (though not all) of the Open Economics datasets and models. Below you can find a link to the main repository containing all our code and data (including that used to run this site), as well as to the main services provided by the project both here and externally.
